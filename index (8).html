Spese ricorrenti

Ogni spesa ha un giorno di scadenza mensile. Il pulsante Applica ora registra il pagamento e scala dal saldo solo nel giorno previsto (o dopo, se scaduta).

—

I giorni con un pallino mostrano le scadenze delle spese ricorrenti. Se una scadenza cade di sabato o domenica e ha l'opzione attiva, viene mostrata spostata al lunedì.

🎯 Budget mensile per categoria

Imposta un tetto di spesa per ogni categoria. La barra si riempie in base a quanto hai già spent questo mese.

🏆 Obiettivo di risparmio mensile

Imposta quanto vorresti riuscire a metter via ogni mese. Lo confrontiamo con il bilancio reale (entrate − uscite) del mese corrente.

🏘️ Spese per immobile/proprietà

Raggruppa automaticamente tutte le spese che condividono lo stesso tag (es. "Lesina", "Roma", "Ururi") indipendentemente dalla categoria. Per associare un movimento a un immobile, inserisci il tag quando lo registri.

💾 Come salvare i tuoi dati

Premi 📦 Esporta JSON in alto a destra (o qui sotto) per scaricare un backup completo con tutti i movimenti, saldi, ricorrenti, storico e snapshot. La prossima volta apri questo file e importa il JSON — trovi tutto.

💰 Saldo iniziale Conto

Il saldo iniziale si imposta solo da qui, non dai movimenti.

💳 Saldo iniziale Carta

Ricarica la carta tramite Trasferimento dal conto.

💵 Saldo iniziale Contanti

I contanti aumentano con il trasferimento Bancomat.

📮 Saldo iniziale PostePay

La PostePay si ricarica solo dai Contanti.

🚨 Allarme saldo minimo

Imposta una soglia per conto: se il saldo scende sotto, vedrai un avviso in home.

💾 Salvataggio

I dati sono salvati dentro questo file HTML. Premi il pulsante per scaricare la versione aggiornata — la prossima volta apri quel file e trovi tutto.

📤 Esporta / Importa dati

JSON — backup completo (movimenti, saldi, ricorrenti, storico, snapshot). Reimportabile senza perdite.
Excel — movimenti formattati con colori e totali, pronto per l'analisi.
CSV — solo i movimenti, formato semplice.

🔧 Ricalcola saldi

Se i saldi sembrano sbagliati, ricalcolali da zero rileggendo tutti i movimenti.
