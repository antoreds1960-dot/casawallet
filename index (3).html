💾 Come salvare i tuoi dati

Premi 📦 Esporta JSON in alto a destra (o qui sotto) per scaricare un backup completo con tutti i movimenti, saldi, ricorrenti, storico e snapshot. La prossima volta apri questo file e importa il JSON — trovi tutto.

💾 Salvataggio

I dati sono salvati dentro questo file HTML. Premi il pulsante per scaricare la versione aggiornata — la prossima volta apri quel file e trovi tutto.

📤 Esporta / Importa dati

JSON — backup completo (movimenti, saldi, ricorrenti, storico, snapshot). Reimportabile senza perdite.
CSV — solo i movimenti, apribile in Excel.

🔧 Ricalcola saldi

Se i saldi sembrano sbagliati, ricalcolali da zero rileggendo tutti i movimenti.
